# enteliscript
Re-write of `enteliWEB` / `enteliSCRIPT`.

## Setup Instructions

### 1. Clone the repository & `cd` into it
```bash
>>> git clone https://github.com/mkaufman2023/enteliweb.git
>>> cd enteliweb
```

### 2. Create a virtual environment & activate it
```bash
>>> python -m venv venv
>>> cd venv/Scripts && activate
```

### 3. Install dependencies
In base directory (`./enteliscript`):
```bash
>>> pip install -r requirements.txt
```

### 4. Run it
```bash
>>> python enteliscript.py
```
